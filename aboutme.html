Technical Definitions

Variables hold the place of a value. You can use the equal sign to assign variables.

Loops make a code run multiple times repeatedly. The types of loops are while loop, for loop, and nested loops.

Conditionals let you change how a program is working using if then, else, and elif statements.

Functions are code that carry out a certain action. Function keywords begin with def.
